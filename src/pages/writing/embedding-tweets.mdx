import { Tweet } from 'astro-embed';

This site embeds tweets as real, interactive cards you can click — the same idea you see
on blogs like [hamel.dev](https://hamel.dev). On a static site hosted on GitHub Pages, the
cleanest way is the [`astro-embed`](https://astro-embed.netlify.app) `<Tweet>` component,
which renders the tweet at build time (fast, and no heavy third-party script at runtime).

For example:

<Tweet id="https://twitter.com/josh_wills/status/198093512149958656" />

Each card is fully clickable and links back to the original post — drop a `<Tweet />`
anywhere in a `.mdx` article and it just works.
